Usage

You can use Button component both as button and a elements

Mantine docs
import React from 'react';
import { Button, ElementsGroup } from '@mantine/core';


function ButtonDemo() {
  return (
    <ElementsGroup position="center">
      <Button onClick={() => console.log('Hello')}>Regular button</Button>
      
      <Button
        component="a"
        href="https://mantine.dev"
        target="_blank"
        variant="outline"
        leftIcon={<ExternalLinkIcon />}
      >
        Mantine docs
      </Button>
    </ElementsGroup>
  );
}

Variants and colors

Button component supports all colors defined in

theme
and has 3 variants:

  • filled (default) – high emphasis button with dark background
  • outline – lower emphasis button with border and transparent background
  • link – lowest emphasis button that looks like regular text

Default Button color is theme.primaryColor, to change color and variant pass color and variant props:

<Button color="red" /> // filled button with red color
<Button variant="outline" /> // outline button with theme.primaryColor color
<Button variant="link" color="indigo" /> // indigo link button

Size

Button has predefined sizes: xs, sm, md, lg, xl. Size prop controls button height, padding and font-size.

<Button size="sm" /> // -> predefined small size
<Button size="xl" /> // -> predefined extra large size

Predefined size values

You can get predefined button heights by importing BUTTON_SIZES:

import { BUTTON_SIZES } from '@mantine/core';

Radius

Radius has predefined values: xs, sm, md, lg, xl which are defined in

theme
, or you can pass number to set border-radius in px:

<Button radius="lg" /> // -> theme predefined large radius
<Button radius={10} /> // -> { borderRadius: 10 }

Full width

Button can take full width of container if you pass fullWidth prop. If button is too large for its container, overflow content will be hidden with ellipsis.

import React from 'react';
import { Button, ElementsGroup } from '@mantine/core';


export function ButtonDemo() {
  return (
    <>
      <div style={{ width: 250 }}>
        <Button fullWidth>Full width button</Button>
      </div>


      <div style={{ width: 180, marginTop: 20 }}>
        <Button fullWidth>Button with overflow</Button>
      </div>
    </>
  );
}

Compose with ElementsGroup

You can compose Button with any other components that support composition feature with

ElementGroup
component.

import React from 'react';
import { Button, ElementsGroup } from '@mantine/core';


function ButtonDemo() {
  return (
    <ElementsGroup position="apart">
      <Button variant="outline">Button 1</Button>
      <Button variant="outline">Button 2</Button>
      <Button variant="outline">Button 3</Button>
    </ElementsGroup>
  );
}

Get element ref

You can get button ref by passing elementRef prop to Button component:

import React, { useRef } from 'react';
import { Button } from '@mantine/core';

function ButtonDemo() {
  const buttonRef = useRef<HTMLButtonElement>(null);
  return <Button elementRef={buttonRef} />;
}

Usage with react-router and other libraries

You can use Button component with react-router-dom or other similar libraries by passing Link as component to Button:

import React from 'react';
import { Link } from 'react-router-dom';
import { Button } from '@mantine/core';

function ButtonDemo() {
  return (
    <Button component={Link} to="/react-router">
      React router link
    </Button>
  );
}

TypeScript support

Usage of some props in typescript require type specification. Button component is a generic and you can pass 2 generic arguments to enhance type support:

// Custom element button
<Button<'a', HTMLAnchorElement>
  component="a"
  onClick={(event) => {}} // -> event will now have correct types
  elementRef={(node) => {}} // node type is set to HTMLAnchorElement
/>

// Custom component button – react-router-dom link
<Button<typeof Link, HTMLAnchorElement>
  component={Link}
  onClick={(event) => {}} // -> event will now have correct types
  elementRef={(node) => {}} // node type is set to HTMLAnchorElement
/>

Usually it is not necessary to use generics and you can avoid it by setting types directly on callbacks:

<Button
  onClick={(event: React.MouseEvent<HTMLAnchorElement, MouseEvent>) => {}}
  elementRef={(node: HTMLAnchorElement) => {}}
/>
