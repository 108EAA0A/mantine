Usage

Badge has 3 variants:

  • light (default) – light background, dark color, no border
  • outline – transparent background, dark color and border
  • filled – dark background, white color, no border

Default Badge color is theme.primaryColor. You can choose any color defined in theme.

import React from 'react';
import { Badge, ElementsGroup } from '@mantine/core';


export function BadgeDemo() {
  return (
    <ElementsGroup position="center">
      <Badge>Light</Badge>
      <Badge variant="outline">Outline</Badge>
      <Badge variant="filled">Filled</Badge>
    </ElementsGroup>
  );
}

Full width

Badge can take full width of container if you pass fullWidth prop. If badge is too large for its container, overflow content will be hidden with ellipsis.

Full width badge
Badge with overflow
import React from 'react';
import { Badge, ElementsGroup } from '@mantine/core';


export function BadgeDemo() {
  return (
    <>
      <div style={{ width: 200 }}>
        <Badge fullWidth>Full width badge</Badge>
      </div>


      <div style={{ width: 120 }}>
        <Badge fullWidth>Badge with overflow</Badge>
      </div>
    </>
  );
}

Custom component

You can change badge root element by passing React element to component prop.

Link badge
$$$ Get lots of money $$$
import React from 'react';
import { Badge, ElementsGroup } from '@mantine/core';


const CustomComponent = ({
  pads,
  children,
  ...others
}: {
  pads: string;
  children: React.ReactNode;
}) => (
  <div {...others}>
    {pads} {children} {pads}
  </div>
);




export function BadgeDemo() {
  return (
    <ElementsGroup position="center">
      <Badge component="a" href="https://mantine.dev">
        Link badge
      </Badge>
      <Badge component={CustomComponent} pads="$$$" variant="filled">
        Get lots of money
      </Badge>
    </ElementsGroup>
  );
}

Compose with ElementsGroup

You can compose Badge with itself and any other components that support composition feature with ElementGroup component.

badge 1
badge 2
import React from 'react';
import { Badge, Button, ActionIcon, ElementsGroup } from '@mantine/core';
import { GearIcon } from '@modulz/radix-icons';


export function BadgeDemo() {
  return (
    <ElementsGroup position="center">
      <Badge>badge 1</Badge>
      <Badge>badge 2</Badge>
      <ActionIcon title="Setting">
        <GearIcon />
      </ActionIcon>
      <Button>Button</Button>
    </ElementsGroup>
  );
}
